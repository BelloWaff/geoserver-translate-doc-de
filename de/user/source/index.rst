GeoServer Benutzerhandbuch
==========================

GeoServer ist eine in Java geschriebene Open Source Software, die es Benutzern gestatet, räumliche Daten zu veröffentlichen und zu bearbeiten. Interoperabel entworfen veröffentlicht er Daten von jeder wichtigen räumlichen Datenquelle durch Benutzung offener Standards.

Dieses Benutzerhandbuch ist ein umfassender Führer in allen Aspekten der GeoServer-Nutzung. Egal ob sie ein Anfänger oder Veteran dieser Software sind, hoffen, wir, dass diese Dokumentation eine hilfreiche Referenz sein wird.
